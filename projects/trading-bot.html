Multi-Strategy Trading Bot

A high-performance algorithmic trading system combining multiple quantitative strategies with a fully vectorised backtesting engine.

Overview

This project implements a modular trading framework that runs momentum, mean-reversion, and risk-off strategies concurrently. A Hidden Markov Model detects market regimes in real time and dynamically allocates capital across the active strategies.

Key Achievements

  • 2 000× backtesting acceleration — replaced iterative row-level simulation with a fully vectorised NumPy engine.
  • Hidden Markov Models for regime detection, enabling dynamic strategy switching between trending and ranging markets.
  • Containerised with Docker for one-command reproducible deployment on any cloud instance.

Architecture

The system is split into three layers: data ingestion (REST + WebSocket feeds), strategy engine (signal generation & portfolio optimisation), and execution (order management & risk controls). Each layer communicates via an internal event bus.

Lessons Learned

Vectorising financial simulations required rethinking the entire data flow to eliminate Python-level loops. Profiling with line_profiler was essential to identify the remaining bottlenecks after the initial rewrite.
